## 🔧 准备工作

### 1. 下载官方版本

从 IDM 官网下载最新的官方安装包，确保软件来源纯净可靠：

```
https://www.internetdownloadmanager.com/download.html
```

### 2. 清理旧版本

> ⚠️ **重要：** 如果之前使用过其他破解补丁或激活工具，请先完全卸载 IDM，并清除相关注册表项和残留文件。

**为什么要清理？**

- 避免激活失败
- 防止软件冲突
- 确保系统环境干净

## 🚀 激活步骤详解

### Step 1：以管理员权限运行 PowerShell

1. 在 Windows 搜索栏中输入 `PowerShell`
2. 右键点击 **"Windows PowerShell"**，选择 **"以管理员身份运行"**

### Step 2：执行激活脚本

本方法使用的是 GitHub 上的开源激活脚本，完全在本地执行，无需担心隐私泄露或恶意代码。

**方法一（短链接）：**

```powershell
irm "https://bit.ly/idm_Activate" | iex
```

**方法二（直接链接）：**

```powershell
irm "https://raw.githubusercontent.com/Astro-Saurav/IDM-Activation-Script/main/IAS.ps1" | iex
```

> 将上述任一命令复制到 PowerShell 窗口中，按 `Enter` 键执行。脚本会自动下载并运行，稍等片刻会弹出一个交互式菜单。

### Step 3：选择激活选项

脚本执行后，会显示以下菜单选项：

```
[1] Activate           - 永久激活 IDM（推荐）
[2] Freeze Trial       - 冻结试用期
[3] Reset Activation   - 重置激活状态
[4] Download IDM       - 下载最新版 IDM
[5] Help               - 查看帮助文档
[0] Exit               - 退出脚本
```

#### 各选项详细说明

| 选项 | 功能 | 适用场景 |
|------|------|---------|
| **`[1] Activate`** | 永久激活 IDM，移除所有使用限制 | **最常用，推荐大多数用户选择** |
| **`[2] Freeze Trial`** | 冻结试用期，始终显示"剩余 30 天试用" | 不想修改注册信息，可无限试用 |
| **`[3] Reset Activation`** | 恢复初始状态，重置激活/试用信息 | 激活出现问题或需要重新激活时使用 |
| **`[4] Download IDM`** | 直接下载最新版本的 IDM 安装包 | 需要获取官方安装包时使用 |
| **`[5] Help`** | 查看帮助文档 | 遇到问题时查阅 |
| **`[0] Exit`** | 退出脚本 | 取消操作时使用 |

> 根据需求输入对应的数字（通常选择 `1`），然后按 `Enter` 确认。部分版本的脚本在选择 `1` 之后可能还需要再按 `9` 才能执行激活，请留意菜单提示。

## 🛡️ 常见问题解答（FAQ）

### Q1：激活后 IDM 提示"假序列号"怎么办？

**原因：** 之前的破解补丁或激活工具残留导致的注册表冲突。

**解决方法：**

1. **完全卸载 IDM**
2. **清理注册表** — 使用 CCleaner 等工具清理相关注册表项
3. **重新安装官方版本**
4. **重新激活** — 再次运行激活脚本，先选选项 `[3]` 重置，再选 `[1]` 激活

### Q2：脚本执行时提示"无法加载文件"错误？

**原因：** PowerShell 执行策略限制，不允许脚本运行。

**解决方法：**

在 PowerShell 中先执行以下命令，临时允许脚本运行：

```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
```

然后再执行激活脚本即可。
